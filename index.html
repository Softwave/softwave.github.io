Commodore 64

I bought a Commodore 64 on eBay. :)

I like old computers and I think this is going to be a whole lot of fun to have. It’ll also be a good way for me to learn assembly programming, specifically of the 6502 sort.

I presently don’t have any software or externals for it, so all I can do right now is play with BASIC, but I intend on buying an SD-card solution for it so that I can use whatever with it and especially so that I can write assembly code on my Mac and compile it with DASM and then move it over and test it on the C64.

Testing out C64 demos on actual hardware should be pretty rad too.

6502 Assembly Programming

I’ve been experimenting with Commodore 64 programming in 6502 assembly, as of late. I’ve been using the wonderful DASM assembler, which I simply installed on OS X through Homebrew. 

processor 6502
	
	org $0801 ; sys2061
sysline:
	.byte $0b,$08,$01,$00,$9e,$32,$30,$36,$31,$00,$00,$00

	org $080d ; sys2061

	jsr $e544 ; clear the screen
	lda #0 
	sta 53281 ; set the screen to black
	lda #0
	sta 53280 ; set the border to black 
	lda #1
	sta 646 ; set the text color to white 
	rts

Haven’t done much with it yet but I definitely plan to keep playing with it… and it’d be nice if I could get my hands on an actual Commodore 64 to play with.
